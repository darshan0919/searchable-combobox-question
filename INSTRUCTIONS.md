# Quick Start

```bash
npm install
npm start
```

Open `src/components/SearchableCombobox/SearchableCombobox.js` and implement milestones **M1 → M4** in order.

Run `npm test` after each milestone to verify progress.

## What to edit

| Edit                                 | Do not edit                                                      |
| ------------------------------------ | ---------------------------------------------------------------- |
| `SearchableCombobox.js` (logic only) | CSS, JSX structure, `mockSearch.js`, `getFocusedIndex.js`, tests |

## Milestone checklist

- [ ] **M1** — Fetch and show results on input change
- [ ] **M2** — Debounce search (300ms)
- [ ] **M3** — Arrow keys + Enter selection
- [ ] **M4** — Select closes list; empty state for no matches

See `README.md` for detailed behavior and hints.
